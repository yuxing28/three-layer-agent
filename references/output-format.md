# 输出格式规范

## 概述

本文件定义了三层思维 Agent 最终输出报告的标准格式。

支持两种输出深度：
- **摘要模式**（默认）：每层核心结论，总报告约 800 字
- **完整模式**（--depth=full）：详细表格和分析，总报告 2000-3000 字

---

## 摘要模式格式（默认）

### Startup 模式摘要报告

```markdown
# 三层思维分析报告

> **分析对象**: [用户的想法]
> **分析日期**: [日期]

## 📊 概览

| 维度 | 核心结论 | 判断 |
|------|---------|------|
| 逻辑层 | [一句话] | 🚦 绿灯/黄灯/红灯 |
| 转化层 | [一句话] | ✅/⚠️/❌ |
| 体验层 | [一句话] | ✅/⚠️/❌ |
| 一致性 | [一句话] | X/10 |

## 🎯 核心结论
[3-5 句话总结分析的核心结论]

## ⚠️ 最大风险
[最需要关注的 1-2 个风险]

## ✅ 最大优势
[这个想法最大的优势是什么]

## 📋 下一步行动
1. [现在应该做的第一件事]
2. [第二件事]
3. [第三件事]

## ❓ 需要补充的信息
[如果有，列出需要进一步了解的信息]
```

## 输出质量自检清单

每个最终报告生成后，需要自检：

- [ ] 是否完整覆盖了三层分析？
- [ ] 每个判断是否有依据？
- [ ] 一致性检查是否执行？
- [ ] 行动计划是否具体可执行？
- [ ] 行动计划的"预期效果"是否有量化数据和推导过程？（不能只写"提升XX"，必须有数字+推导）
- [ ] 语言是否清晰易懂，避免专业术语堆砌？
- [ ] 是否指出了信息不足的地方？
- [ ] 是否给出了明确的判断（而不是"可能"、"或许"）？
- [ ] 完整模式最终报告是否在2000-3000字以内？（不含分步分析的详细内容）
