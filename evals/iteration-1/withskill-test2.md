# withskill-test2: Edge Case - Minimal Information Input

## User Message
> 我想做个AI相关的创业项目，还没想好具体方向，你帮我分析分析？

## Mode Detection
- Contains "我想做" + "创业" -> **Startup mode**
- Contains "还没想好具体方向" -> Information is extremely insufficient

## Step 0: Minimum Information Check

| Required Item | Status | Provided? |
|---|---|---|
| 解决什么问题（痛点） | Missing | No - user explicitly says "还没想好具体方向" |
| 给谁用（目标用户） | Missing | No |
| 当前状态 | Partially implied | "还没想好" suggests pure idea stage, no product |
| 竞品参考 | Missing | No |

**Verdict**: Information insufficient - both mandatory items (痛点, 目标用户) are missing. Cannot proceed to analysis.

---

## Skill Response Output

好的，AI创业方向是个大话题，我需要了解一些关键信息，分两轮来聊——先聊必须搞清楚的，再聊建议补充的。

**第一轮，两个必须回答的问题：**

1. **你想解决什么问题？** 不需要很精确，但大致方向要有——比如"企业招聘效率低"、"普通人学编程太难"、"某某行业的XX流程很痛苦"。哪怕是"我觉得XX场景用AI能做得更好"也行，我需要知道你的AI能力准备落在哪个痛点上。

2. **大概想给谁用？** 企业用户（To B）还是个人用户（To C）？有没有某个行业或人群是你比较熟悉的？比如"中小电商卖家"、"内容创作者"、"制造业工厂"等。

回答这两个之后，我再问第二轮建议补充的问题（比如当前状态、竞品参考、预算和资源等）。
